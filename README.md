# George nvim

## Keymaps - in progress
```
<>x - close
<>c - copy line
<>cs - clear search

<>sv - split vertical
<>sh - split horizontal
<>hjkl - move between splits

<>e - open file explorer

// insert mode
jk - save and esc
jj - insert ()
kk - insert {}

// visual mode
<>J K - move line up or down
```
